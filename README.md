# Task Master

## Overview

The **Task Master** is a Spring Boot application that allows users to register, log in, and for a task tracking and managing application that facilitates collaboration and organization within teams or projects. It utilizes **JWT** for secure token-based authentication.

## Features

### 1. **User**
- `POST /api/register`: Register a new user.
- `POST /api/login`: Authenticate a user and issue a JWT token.

## Technologies Used

- **Spring Boot**: Core framework.
- **Spring Security**: JWT-based authentication.
- **Maven**: Build tool.
- **My SQL**: Stores user data and preferences.

## Running the Application

### Prerequisites

- **Java 21** or later
- **maven** installed


## Testing

Use **Postman** or **cURL** to validate API functionality.
